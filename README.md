# Análise de Anúncios de Veículos

# Descrição do Projeto

Este projeto foi desenvolvido utilizando Python, Streamlit e Plotly Express. O objetivo é criar uma aplicação web interativa para realizar uma análise exploratória de um conjunto de dados de anúncios de veículos.

A aplicação permite visualizar informações do conjunto de dados de forma intuitiva, auxiliando na compreensão da distribuição e da relação entre algumas variáveis.

# Funcionalidades

- Exibe um cabeçalho com a descrição da aplicação.
- Permite gerar um histograma da quilometragem (odometer) dos veículos.
- Permite gerar um gráfico de dispersão mostrando a relação entre quilometragem (odometer) e preço (price).
- Utiliza gráficos interativos criados com Plotly Express.
- Interface desenvolvida utilizando Streamlit.

# Tecnologias Utilizadas

- Python
- Pandas
- Plotly Express
- Streamlit

# Como executar o projeto

1. Clone o repositório.
2. Instale as dependências:

```bash
pip install -r requirements.txt
```

3. Execute a aplicação:

```bash
streamlit run app.py
```

4. Abra o navegador no endereço exibido pelo Streamlit (geralmente http://localhost:8501).
