# Análise de Anúncios de Veículos

## Descrição

Este projeto consiste em uma aplicação web desenvolvida com Streamlit para análise exploratória de um conjunto de dados de anúncios de veículos.

## Funcionalidades

- Histograma da quilometragem dos veículos.
- Gráfico de dispersão entre preço e quilometragem.
- Visualizações interativas com Plotly.

## Tecnologias

- Python
- Pandas
- Plotly Express
- Streamlit
